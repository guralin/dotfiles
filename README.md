# guralinのvimrc保存場所


## vimの中

- colorscheme
- dein.vim
- NEADTree

## 使用方法

1. `cd ~/`
1. `git clone https://github.com/guralin/dotfiles.git`
1. `sh start.sh`

カラースキーム'<名前>'が見つかりませんと言われたときはもう一度`sh start.sh`
